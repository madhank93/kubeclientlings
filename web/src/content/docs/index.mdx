## What is KubeClientlings?

KubeClientlings is a set of small, broken-on-purpose Go programs that talk to a
real Kubernetes cluster — [rustlings](https://rustlings.rust-lang.org/) for
[client-go](https://github.com/kubernetes/client-go). An interactive terminal UI
re-runs each exercise as you save, and every fix is verified against a local
[kind](https://kind.sigs.k8s.io/) cluster, not mocks.

## Start in three commands

```sh
git clone https://github.com/madhank93/kubeclientlings && cd kubeclientlings
mise run up      # create the local kind cluster
mise run watch   # launch the interactive TUI
```

Prefer zero local setup? Run it in the browser on the
[iximiuz Labs playground](https://labs.iximiuz.com/playgrounds/kubeclientlings).

See [Getting started](/getting-started/) for the full setup.
